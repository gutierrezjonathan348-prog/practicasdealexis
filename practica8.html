Contadores en CSS (Lineales, Jerárquicos y Anidados)

Descripción de la práctica

En esta práctica se analizan diferentes formas de utilizar los contadores en CSS, los cuales permiten numerar elementos de forma automática sin necesidad de escribir los números manualmente en el HTML.

Evidencias de la práctica

Explicación del código

Código 1: Contador lineal simple
Se utiliza counter-reset para iniciar un contador y counter-increment para aumentarlo cada vez que aparece un elemento h2.

Código 2: Secciones y subsecciones dinámicas
Se manejan dos contadores: uno principal y uno secundario, generando numeraciones como 1.1, 2.1, etc.

Código 3: Contadores anidados
Se utiliza la función counters() para crear jerarquías automáticas como 1.1.1 o 2.3.1.

Conclusión

Los contadores en CSS permiten automatizar la numeración de contenido, haciendo el código más limpio, organizado y profesional.
