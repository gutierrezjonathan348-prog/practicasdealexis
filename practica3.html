Práctica 3

Tablas en HTML

Descripción de la práctica

En esta práctica se aprende a crear tablas utilizando HTML para organizar información en filas y columnas. Las tablas son una herramienta muy útil cuando se desea mostrar datos de manera estructurada y facilitar su interpretación.

La estructura principal se crea mediante la etiqueta <table>, la cual contiene todos los elementos de la tabla.

Las filas se generan utilizando la etiqueta <tr>, mientras que las celdas donde se almacena la información se crean mediante la etiqueta <td>. Para los encabezados se utiliza la etiqueta <th>, la cual permite destacar la información principal de cada columna.

Además, pueden emplearse atributos como border para agregar bordes y mejorar la presentación de los datos. También es posible crear tablas simples y tablas más complejas para organizar información de diferentes maneras.

Objetivo: Comprender la estructura y funcionamiento de las tablas en HTML para organizar datos de manera clara, ordenada y profesional dentro de una página web.

Etiquetas utilizadas

Etiqueta Función
<table> Define la tabla completa.
<tr> Crea una fila.
<th> Crea un encabezado para una columna.
<td> Contiene los datos de cada celda.
border Agrega un borde visible a la tabla.

Evidencia de la práctica

Conclusión

Las tablas permiten organizar grandes cantidades de información de forma clara y ordenada. Gracias a las etiquetas propias de HTML es posible crear estructuras fáciles de interpretar por los usuarios, mejorando la presentación de datos y facilitando la lectura dentro de cualquier sitio web.
