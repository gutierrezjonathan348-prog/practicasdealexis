Práctica 4

Formularios en HTML

Descripción de la práctica

En esta práctica se estudia la creación de formularios utilizando HTML para recopilar información de los usuarios. La estructura principal se desarrolla mediante la etiqueta <form>, la cual contiene todos los elementos del formulario.

Se utiliza la etiqueta <input> en diferentes tipos, como text para capturar texto, password para ocultar caracteres, radio para seleccionar una única opción, checkbox para seleccionar varias opciones, submit para enviar la información y reset para restablecer los datos del formulario.

También se emplean las etiquetas <textarea> para introducir textos largos y <select> junto con <option> para crear listas desplegables.

Objetivo: Aprender a diseñar formularios funcionales que permitan capturar y gestionar información de manera organizada dentro de una página web.

Evidencias de la práctica

Conclusión

Los formularios representan uno de los elementos más importantes del desarrollo web, ya que permiten la interacción entre el usuario y la página. Gracias a los diferentes tipos de controles disponibles en HTML es posible recopilar información de manera eficiente, organizada y sencilla, siendo la base de sistemas como registros, inicios de sesión, encuestas y formularios de contacto.
