Práctica 2

Listas en HTML

Descripción de la práctica

En esta práctica se trabaja con la creación y organización de listas utilizando HTML. Las listas permiten presentar información de una forma ordenada y fácil de comprender, siendo uno de los elementos más utilizados dentro del desarrollo web.

La etiqueta <ol> permite crear listas ordenadas mostrando automáticamente una numeración para cada elemento, mientras que la etiqueta <ul> crea listas no ordenadas utilizando viñetas.

Dentro de ambos tipos de listas se utiliza la etiqueta <li>, la cual representa cada elemento individual de la lista.

También se emplean las listas de definición mediante las etiquetas <dl>, <dt> y <dd>, donde <dl> define la lista completa, <dt> representa el término y <dd> contiene la descripción correspondiente.

Estas etiquetas ayudan a organizar información de forma clara, mejorando la presentación y facilitando la lectura para el usuario.

Objetivo: Aprender a utilizar correctamente los diferentes tipos de listas que ofrece HTML para organizar información de manera estructurada dentro de una página web.

Etiquetas utilizadas

  • <ol> → Crea listas ordenadas.
  • <ul> → Crea listas con viñetas.
  • <li> → Representa un elemento de la lista.
  • <dl> → Define una lista de definiciones.
  • <dt> → Indica el término.
  • <dd> → Muestra la descripción del término.

Evidencia de la práctica

Conclusión

Las listas son herramientas fundamentales dentro de HTML porque permiten organizar el contenido de una página de manera clara y ordenada. Gracias al uso de listas ordenadas, no ordenadas y de definición, es posible presentar información de forma más comprensible, mejorando tanto la estructura del documento como la experiencia del usuario al navegar por el sitio web.
